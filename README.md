# SQL-End-to-End-Portfolio-Project
End-to-End SQL data analysis project using MySQL and an Online Store dataset.
## Project Overview
This project analyzes an online book store database using MySQL.

## Dataset
Online Book Store Dataset

## Tools
- MySQL Workbench
- SQL

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- LIMIT
- INNER JOIN
- Aggregate Functions
- COALESCE

## Business Questions Solved
- Find customers with multiple orders
- Most frequently ordered book
- Top expensive fantasy books
- Total books sold by each author
- Customers spending above a threshold
- Highest spending customer
- Remaining stock after sales

## Skills Demonstrated
SQL, Data Analysis, Business Insights
